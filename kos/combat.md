# Combat Rules

## Banned Tactics

* No cheating. This is an instant ban, full war DQ, and gang strikes.
* No crouch shooting.
* No aqua peeking.
* No off-Q peeking.
* No drug shooting.
* No shooting from rooftops, mountains, or high ground.
* No helmets of any kind.

## The "W8" Call

Calling "W8" requires the entire team to stop playing immediately.

{% hint style="warning" %}
If anyone continues playing after a "W8" call, the round is awarded to the opposing team.
{% endhint %}

## Violations

| Offense | Result |
| ------- | ------ |
| First violation | Round Loss |
| Second violation | War DQ |

## RDM in Wars

Killing someone outside of the KOS is RDM.

| Offense | Result |
| ------- | ------ |
| RDM x1 | Round DQ |
| RDM x2 or more | War DQ |
