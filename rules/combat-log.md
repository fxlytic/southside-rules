# III. F8 / Combat Log

**Combat logging** is defined as **purposely leaving a scene or closing the game to avoid an unwanted outcome.**

## The Rule

When engaged in active roleplay, players should **not disconnect** from the server until the scenario has **completely ended**.

If you are a victim of [RDM](rdm.md), [Fail RP](fail-rp.md), or any other invalid situation, you must:

1. **Remain in the scene and city** until it is over.
2. **Make a report in-game.**

{% hint style="danger" %}
You **cannot combat log** because someone else broke the rules. Stay in the scene — you'll be **reimbursed** if the other person violated a server rule.
{% endhint %}

## Crashes

If you crash, you **must have your crash file** to prove you actually crashed.

## Examples of Combat Logging

* F8ing / disconnecting **while being robbed**.
* F8ing / disconnecting **during a chase**.
* F8ing / disconnecting **during any scene** in general, or during an **admin sit**.

{% hint style="info" %}
Getting wronged is frustrating, but the system works **only if you stay**. Leave the scene and you forfeit your reimbursement — and may be punished yourself.
{% endhint %}
