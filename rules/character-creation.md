# I. Character Creation

Your character is your identity in SouthSide. Put effort into it.

## Rules

* **Realistic names only.** Character names must resemble real-world names.
* **No defaults.** Your character must have some type of customization — not a default appearance.
* **Names must follow TOS guidelines.**
* **No troll characters.** Put effort into your character or you'll be asked to change it. This can result in an **FRP ban**.
* **Peds must match.** Females must be on female peds, males must be on male peds.

{% hint style="info" %}
We are **LGBTQ friendly** — that roleplay is allowed! The ped-matching rule is about realism, not identity.
{% endhint %}

## 👕 Clothing Restrictions

**No "100k or die" clothing allowed**, including:

* Face paint
* Hockey mask / troll mask
* Black eyes
* No hand
* Bare feet
* Colored tattoos

{% hint style="warning" %}
Low-effort or troll characters can lead to a forced change **and** an FRP ban. When in doubt, make it realistic.
{% endhint %}
